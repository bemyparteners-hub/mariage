# 📸 Images du site

Déposez vos photos dans ce dossier. Pour que je puisse les intégrer
automatiquement aux bons endroits, voici les **noms de fichiers suggérés**
(le format `.jpg`, `.jpeg`, `.png` ou `.webp` fonctionne) :

| Nom de fichier             | Emplacement sur le site                          |
|----------------------------|--------------------------------------------------|
| `hero.jpg`                 | Grand visuel d'accueil (plein écran)             |
| `grand-jour.jpg`           | Section « Notre grand jour »                     |
| `histoire-1.jpg`           | Timeline — Les premiers messages                 |
| `histoire-2.jpg`           | Timeline — La rencontre                           |
| `histoire-3.jpg`           | Timeline — Chavanoz                               |
| `histoire-4.jpg`           | Timeline — Corbas                                 |
| `histoire-5.jpg`           | Timeline — Tonka                                  |
| `histoire-6.jpg`           | Timeline — Le PACS                                |
| `histoire-7.jpg`           | Timeline — La maison à Villefontaine              |
| `demande.jpg`              | Section « La demande en mariage » (Aix-les-Bains)|
| `temoin-alexane.jpg`       | Témoin — Alexane                                  |
| `temoin-valerie.jpg`       | Témoin — Valérie                                  |
| `temoin-antoine.jpg`       | Témoin — Antoine                                  |
| `temoin-justine.jpg`       | Témoin — Justine                                  |
| `photo-1.jpg` … `photo-6.jpg` | Galerie photos                                |

**Pas besoin de respecter tous ces noms** : déposez simplement ce que vous
avez, dites-moi quel fichier va où, et je m'occupe du reste.

> Astuce : pour récupérer une image depuis votre Google Sites, faites un
> clic droit dessus → « Enregistrer l'image sous… », puis glissez-la ici.
